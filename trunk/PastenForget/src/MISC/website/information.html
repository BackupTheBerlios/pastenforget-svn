Information

Paste 'n' Forget is a downloader for One-Click-Hoster like Rapidshare and streaming sites like Youtube. The program automates the different download-steps for Free-User. The downloads of each service are organized in a queue.

Each developer work for the project in his own free time. Most of us are students in computing. We want to gain experience with help of the project.

Requirements :Internet connection, Java Runtime Environment Version 6
