2008-Nov-03 20:34PnF-v0.400-Alpha released!
Today we release "Paste 'n' Forget" in version 0.400 (Alpha). We add some useful features in this release and fix some bugs. Enjoy PnF and don't be afraid of bugs ;)

New features:
  • Icons in the toolbar
  • IRC download (still testing)
  • Captcha input
  • RSDF-Container support
  • Multilanguage support
  • Log-View
  • Faulty downloads are put to the end of queue, so that the next download won't be hold up
  • More than one download can be stopped or canceled
