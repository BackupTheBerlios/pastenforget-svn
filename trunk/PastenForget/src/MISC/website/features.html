Features (v0.400 Alpha)

  • Operating System independent (programmed in Java)
  • Multilanguage support: English, German (at the moment)
  • Supported Hoster (only Free-User): Rapidshare, Uploaded, Megaupload
  • Supported Streaming Sites: YouTube and more!
  • Search- and Linkfilterfunction for ddl-warez.org
  • PNF-Download: Save your links in a textfile and add the downloads in one action
  • RSDF-Container support
  • IRC-Download (Still testing)
  • Searchfunction for packetnews.com
